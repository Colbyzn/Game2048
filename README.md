#### 使用说明
通过键盘上的上下左右键，实现棋盘格的移动。

1.  **←   向左移动**
2.  **↑   向上移动**
3.  **→   向右移动**
4.  **↓   向下移动**

#### 制作该游戏需要准备的技能
1.  **游戏逻辑** —— 解决办法：亲自玩一下2048游戏，了解游戏的规则；
2.  **技术** —— HTML、CSS、JavaScript、JQuery 。
